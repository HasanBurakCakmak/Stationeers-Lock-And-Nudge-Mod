# Stationeers Lock and Nudge Mod

Lock and nudging mechanic created for building in the Stationeers game using C# and Harmony lib. 

## Requirements
* BepInEx 2.4.xx
* Stationeers Launchpad

## How to Use
* **Lock Object:** `H`
* **Nudge Horizontal:** `Arrow Keys`
* **Nudge Vertical:** `PageUp/PageDown`
