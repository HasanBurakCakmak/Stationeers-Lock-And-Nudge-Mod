# Stationeers Lock and Nudge Mod

Lock and nudging mechanic created for building in the Stationeers game using C# and Harmony lib. 

## Requirements
* BepInEx 2.4.xx
* Stationeers Launchpad

## How to Use
* **Lock Object:** `H`
* **Nudge Horizontal:** `Arrow Keys`
* **Nudge Vertical:** `PageUp/PageDown`

## Installation
1. Ensure you have BepInEx 2.4.xx and Stationeers Launchpad installed.
2. Download the latest `.dll` file from the [Releases] tab.
3. Place the `.dll` file into your `BepInEx/plugins` folder inside your Stationeers game directory.
4. Launch the game!
